### Getting Started
1. Download Latest version of [Git for Windows](https://git-scm.com/downloads/win) and [Node.js](https://nodejs.org/en/download)
2. To verify installation, open cmd and run:
```cmd
git -v
node -v
npm -v
```
3. Clone the Repository
```cmd
git clone https://github.com/your-username/chatbot.git
cd chatbot
```
4. Install npm
```cmd
npm install
```
5. Sign up or log in to [Cohereâ€™s dashboard](https://dashboard.cohere.com/welcome/login) and Generate or locate your API key. Then open the .env file and add your API key like this:
```cmd
REACT_APP_API_KEY=YourAPIKeyHere
```
6. Run the Dev Server in cmd
```cmd
npm run dev
```
7. Open It in Your Browser
```cmd
http://localhost:5173
```
